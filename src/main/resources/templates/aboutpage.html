About Us

Welcome to my workout app!

This is my project for Web Programming and here is everything used:

  • Postgres database, instead of local H2 database. This is due to login/register form. There is a functionality for registering new users to the database. H2 databases are not good for persisting data across application restarts. Postgres is configured using PgAdmin4.
  • Login/register form that registers new users via the User model and lets the same registered users sign in via the login form. The user is stored and populated via Postgres.
  • Exercise model which is populated via the DataInitializer in the config package. The exercises are populated locally and displayed dynamically in the HTML.
  • BMI Calculator, a separate functionality that lets the user see his/her Body Mass Index. When the index is shown, the user is automatically placed in a weight category. There is also a picture shown so that the user has a general idea of how their weight is impacting their health.
